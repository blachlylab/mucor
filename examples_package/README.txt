This package of examples is intended to supplement MuCor: Mutation Correlation and Aggregation.
Execute "setup.sh" to:
1. download the required references 
2. reformat the references
3. place symbolic links to the example data in each of the example project directories
Establishes examples 1, 2, and 3

Execute "depthgauge_setup.sh" to: 
1. download example BAM files [totalling ~110 GB]
2. place symbolic links to the example data in the example project directory
Establishes example 5
